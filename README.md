# Retail Exception Dashboard

A modern retail exception management dashboard built using .NET MAUI Blazor Hybrid.

---

## Overview

This application provides a simple and interactive UI to view and manage retail exceptions such as inventory mismatches, pricing issues, and vendor discrepancies.

---

## Features

- View exception list in a structured table
- Search exceptions dynamically
- Filter by status (Open, InReview, Resolved, etc.)
- Sort by Type, Item, Store, and Status
- Resolve exceptions via action button
- Status badges with color indicators

---

## Tech Stack

- .NET MAUI
- Blazor Hybrid
- C#
- Razor Components

---

## UI Highlights

- Responsive table layout
- Interactive sorting with arrow indicators
- Colored status badges
- Clean and minimal design

---

## How to Run

1. Open the solution in Visual Studio
2. Select **Windows Machine**
3. Click **Run (▶)**
